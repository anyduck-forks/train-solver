# Train Solver

Built as a term project for MMDO (Mathematical Methods of Operations Research).

The 3D shaders and UI design were inspired from [Remix3](https://remix.run/) ([source code](https://github.com/remix-run/remix/commit/2c0ef67220714e1005162be4acdb91fbf355c664))

### Development
```bash
wasm-pack build --target web --release
pnpm run dev
```

### Build

```bash
wasm-pack build --target web --release
pnpm run build
```
